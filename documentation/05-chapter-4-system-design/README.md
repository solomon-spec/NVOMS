# Chapter 4 Structure

This chapter is arranged according to the guideline and the current project needs.

- `01-overview-and-design-goals/`
  overview, design goals, performance, scalability, security, availability, usability, interoperability
- `02-software-architecture/`
  layered architecture, service design, architecture diagrams
- `03-subsystem-decomposition/`
  subsystem decomposition write-up and related diagrams
- `04-database-design/`
  schema design, verification, ERD code, and database explanation
- `05-deployment-diagram/`
  deployment nodes, runtime environment, and deployment visuals
- `06-user-interface-design/`
  screens, wireframes, and UI explanation
- `07-system-integration/`
  DHIS2, FHIR, SMS gateway, weather API, and data exchange planning
- `08-security-design/`
  authentication, authorization, audit logging, and data protection
- `09-design-verification/`
  traceability and design verification against requirements
