# Frontend Area

This area is for the web-based client structure.

Recommended structure:

- `web/src/app/`
  application routes or page entry points
- `web/src/components/`
  reusable UI components
- `web/src/features/`
  feature-based screens and logic
- `web/src/services/`
  API calls and client-side service wrappers
- `web/src/shared/`
  shared frontend-only utilities and types
- `web/src/styles/`
  styling and theme files
- `web/public/`
  static assets
- `web/tests/`
  unit and integration test folders
