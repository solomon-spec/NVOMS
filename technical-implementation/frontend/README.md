# Frontend Area

This area is for the web-based interface of NVOMS.

Recommended structure:

- `web-portal/app/`
  application routes or page entry points
- `web-portal/components/`
  reusable UI components
- `web-portal/features/`
  feature-based screens and logic
- `web-portal/services/`
  API calls and client-side service wrappers
- `web-portal/styles/`
  styling and theme files
- `web-portal/public/`
  static assets
