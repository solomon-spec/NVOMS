# Template Components

This frontend imports reusable components from the free TailAdmin Next.js template.

Rules for this integration:

- Use template source only as reusable components and support assets.
- Do not copy or depend on the template demo pages or route structure.
- Prefer local imports under `web/src/components/...`.
- Build application pages and feature flows from scratch in this repository.

Imported source basis:
